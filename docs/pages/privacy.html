Power Editor - privacy policy

Confidentiality policy document on the use of confidential data by the application. This document describes what data is used by the application and for what purposes.

User information

Power Editor does not collect user information. App can use your device's local account or Microsoft account to sync and store your settings and preferences.

Storage usage

Power Editor requires full access to the file system for normal work, as the application can edit folders and files outside of the application's internal files. If the user does not grant permission to the file system, the application will not be able to edit the files as expected. By default, the application has access to the system libraries (Music, Pictures, Documents, Videos, Removable Devices). If you use Xbox, please save and work with files in these libraries because there is no support for file system permission on Xbox.

Microphone

Power Editor can use the microphone to recognize speech, voice commands, and web search(Bing, Google, DuckDuckGo, Yahoo, Wolfram, Wikipedia).

Package query

Power Editor can collect data about installed packages to run extensions.

Access to user marginal notes

The application can create, manage, edit user notes and store them in a Microsoft account. The application does not transfer notes and all information related to them to third parties (it is available only for you).

Document changes

The document may change from time to time because the application may request more data depending on updates and new features.
